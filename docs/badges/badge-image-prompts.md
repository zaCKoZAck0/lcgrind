# Badge Image Prompts

Gemini (Imagen / Nano Banana) text-to-image prompts for the 19 lcgrind achievement badges.

Visual language: NEOBRUTALISM. Thick uniform 4px pure-black outlines, hard-edged solid offset drop shadows (no blur), flat saturated fills, no gradients, no 3D, no gloss. Sticker/vector look, transparent background, 1:1.

Tier metals (frame + ribbon banner): Bronze `#CD7F32` · Steel/iron grey `#7C8794` · Silver `#C0C7CF` · Gold `#FACC15` · Platinum cool white-violet `#DDE3F0`. Frame shape is a hexagonal medal for every badge.

### valuable-contributor

```text
A flat neobrutalist achievement badge icon for a "Valuable Contributor" award.

Subject: a bold geometric stack of coins topped by a cut gem at the center, simplified into clean flat shapes like a Lucide line icon turned solid. The stack sits inside a chunky hexagonal medal frame.

Style: neobrutalism. Thick uniform 4px pure-black outlines on every shape. A hard-edged solid drop shadow offset down-right (no blur, no gradient). Flat saturated fills only — no shading, no glow, no 3D, no texture. Color: emerald-green gem (#10B981), accents in bright gold coins; medal frame in bronze (#CD7F32) with a black border; one small ribbon banner under the medal reading "COMP".

Composition: single centered badge, square 1:1, generous padding, sticker-like, crisp vector look, fully symmetrical.

Background: transparent.

Negative: no gradients, no glossy reflections, no realistic metal, no emoji, no photographic texture, no soft shadows, no clutter, no extra text.
```
